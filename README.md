# image2webp

** Grab the images in the page and convert them to webp format.**

**support**：
1. jpg -> webp
2. jpeg -> webp
3. gif -> webp
4. png -> webp

- 此工具只能以命令行运行。
- windows下可以直接运行cmd.bat
- 转换后的图片在webp目录下
- 请注意：Linux下一定要给webp目录可写入权限。
